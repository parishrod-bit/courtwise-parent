🚀 Quick Launch

Make CourtWise easy to open every game

This HTML app cannot always create its own phone shortcut automatically. Open it in the actual phone browser, not a file/email preview.

Android / S24 Ultra

  1. Open CourtWise in Chrome or Samsung Internet.
  2. Tap the browser menu.
  3. Choose Add to Home screen or Create shortcut.
  4. Name it CourtWise Parent and tap Add.

iPhone

  1. Open CourtWise in Safari.
  2. Tap Share.
  3. Tap Add to Home Screen.
  4. Name it CourtWise Parent and tap Add.

Important: If buttons do not respond on iPhone, you are likely viewing the file in a preview. Reopen it in Safari.

About

CourtWise Parent

Created, developed, and owned by Rod Parish.

v0.3.1 RELEASE • Phone-first athlete game tracker.

Purpose: Track one athlete quickly while still watching the game.

Data: Auto-saved locally on this device and exportable as JSON/CSV.

How to Use

During play: Use the big buttons only. Tap Undo if the last tap was wrong.

Save Game: After the game, enter opponent/final score and save. Saved games build history and season averages.

No clock: The live Q/time section was removed because parents should not have to run a clock.

Photo: Upload/take a photo, then move and zoom it before saving.

Data: Auto-save happens after every tap. Use Backup JSON often, especially before switching phones.

iPhone: Open in Safari. If it opens like a picture and buttons do not work, you are in a preview, not the browser app.
