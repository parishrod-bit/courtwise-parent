About

CourtWise Parent

Created, developed, and owned by Rod Parish.

v0.3.2 RELEASE • Phone-first athlete game tracker

Purpose: Track one to five athletes quickly while still watching the game.

Design principle: Big taps first, history second, complexity hidden.

Data: Stored locally in this browser unless exported/restored.

How to Use

Pick player: Tap the player card at the top. All stat buttons apply to the active player.

During play: Use the big buttons only. Tap Undo if the last tap was wrong.

Shooting: +2 and MISS 2PT track 2-point percentage. +3 and MISS 3PT track 3-point percentage. FG shows total shooting.

Save Game: Saves all tracked players with opponent, final score, date, and season.

Clear Game: Clears today’s live stats only. Saved history remains.

Backup: Export JSON before switching phones, browsers, or clearing browser data.

iPhone: Open the hosted link in Safari and Add to Home Screen. Do not use downloaded HTML files.
