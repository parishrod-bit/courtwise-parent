How to Use

First-time setup

  1. Open CourtWise from the hosted GitHub Pages link.
  2. Use 🚀 Open to add it to the Home Screen: Safari on iPhone; Chrome/Samsung Internet on Android.
  3. Tap ☰ Menu in portrait or Setup in landscape.
  4. Enter 1–5 players.
  5. Select exactly one Primary athlete.
  6. Add each player photo if desired, crop/zoom it, then tap Save Crop.
  7. Tap Save Setup.

During the game

  1. Tap the player card for whoever should receive the next stat.
  2. The orange card is the active player.
  3. Tap stat buttons. The stat goes only to the active player.
  4. Use Undo if the last tap was wrong.

Shooting buttons: Made and missed shots are paired for speed: +2 with MISS 2PT, +3 with MISS 3PT, FT with FT MISS.

Save Game: enter opponent and final score at the end. CourtWise saves stats for every configured player in that game.

Reports: choose one player or All Players, then choose All Saved Games or Selected Games. You can Print/PDF, Email, or Save Local Report.

Player photos: Each player can have their own photo. The Primary player photo appears in the header and main player card. Reports use the selected player photo when reporting one player.

Portrait vs landscape: Portrait uses ☰ Menu to save space. Landscape keeps the full top controls visible.

Close App: tap Close App / End Day when finished. CourtWise autosaves current game, player, and history data unless you clear/reset it.

Backup: use Backup JSON before changing phones, clearing browser data, or testing a new version.
