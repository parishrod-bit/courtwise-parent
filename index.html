About

CourtWise Parent

Created, developed, and owned by Rod Parish.

v0.3.4 RELEASE • Phone-first athlete game tracker

Purpose: Track one to five athletes quickly while still watching the game.

Design principle: Big taps first, history second, complexity hidden.

How to Use

First-time setup

  1. Tap Setup.
  2. Enter 1–5 players.
  3. Select one player as Primary.
  4. Tap Save Setup.

During the game

  1. Tap the player card for whoever you want to track.
  2. The orange card is the active player.
  3. Tap stat buttons. The stat goes only to the active player.
  4. Use Undo if the last tap was wrong.

MISS 2PT / MISS 3PT: use these so CourtWise can calculate 2PT%, 3PT%, and total FG%.

Save Game: enter opponent and final score at the end. CourtWise saves stats for all players tracked that game.

Primary Athlete: appears first in reports and gets the header/photo focus.

Backup: use Backup JSON before changing phones, clearing browser data, or testing a new version.
