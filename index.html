About

CourtWise Parent

Created, developed, and owned by Rod Parish.

v0.3.3 RELEASE • Multi-player parent tracker

Purpose: Track 1–5 athletes quickly while still watching the game.

Data: Stored locally in this browser unless exported/restored.

How to Use

Pick player first: Tap a player card at the top. The highlighted card receives the next stat.

During play: Use the big buttons only. Tap Undo if the last tap was wrong.

Misses: MISS 2PT and MISS 3PT are separate so CourtWise can calculate 2PT%, 3PT%, and total FG%.

Save Game: Saves stats for every tracked player, plus opponent and final score.

Primary athlete: The primary player appears in the header and leads reports.

Backup: Export JSON before switching phones or clearing browser data.

iPhone: Use the hosted GitHub Pages link in Safari, then Add to Home Screen.
